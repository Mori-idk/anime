# anime
Anime Data Integration (SSIS)
Automatyzacja importu danych o anime z CSV do SSMS przy użyciu SSIS

Kluczowe funkcjonalności:

Konwersja typów danych.
Obsługa formatowania regionalnego.
Przekierowanie błędnych rekordów (np. "Unknown episodes") do osobnego pliku logu.
